Mike's Calzones Redesign

This project is about redesigning a user interface that currently exists. I chose the restaurant and wesbite MikesCalzones.com to redesign.
First, I will discuss the problems with the interface. Then, I will go showing the lo-fi and hi-fi prototypes of the website. Finally, I will redesign the website.

Old Interface

Design Concerns
  • Color contrast could be improved using the same signature colors
  • Information grouping is confusing and is hard to quickly parse for the user
  • Text on home page is seemingly unscannable
  • Navigating to other parts of the other parts of the site isn't obvious and is subtle
  • Hours aren't centered/aren't in line with the scan
  • Hours are hard to parse for a specific day
  • Content seems randomly placed on the page
Accesibility Issues
Mike’s Calzones has 1 error, 9 contrast errors, 1 alert, 4 features and 3 structural elements. The contrast errors are very low contrast errors, and the alert is an empty heading. Screen reader reads the title on the home page and the hours, but not the description on the side or how to get to other pages.

Lo-Fi Prototyping

The Lo-Fi wireframes shows the Website View, the Phone View and the Tablet View respectively.
  1. Color Contrast is now much more attractive, while still using the same key colors
  2. Menu is easily accesible and it is easy to find
  3. Text is easy to scan now. The site is very easy to parse quickly
  4. Hours are easily found, and the hours are cleanly displayed
  5. Operating around the site is very simple, buttons are obviously links

Visual Design Guide

The important colors for the redesigned website are the same that are used on the old website. Here are the following signature colors: Cobalt Blue: #1400A2, Orange: #D66824, Light Blue: #597198, White: #000000, and Grey: #A29797. All of the buttons should on hover turn Grey, but have original color Cobalt Blue with an Orange Border. Title should follow the same schematics, but it does not change color on hover. The delicious has font family Bodoni MT Black, Buttons have font family Arial, and the title has font family Century.
